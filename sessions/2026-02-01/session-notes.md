# Part 1: 语法强化与角色扮演 (Grammar & Role-play)

## 提出的问题

### 问题 1: Subjunctive Past (虚拟语气过去时)
- **学生的问题**: 区分 `had done` (过去完成) 与 `would have done` 的搭配。
- **初始理解**: 在 If 从句中出现了 `had not have` 的错误，以及主句否定词位置 `would have not` 的混淆。
- **给出的解释**: 
    1. If 从句结构必须是 `had` + 过去分词 (have 的分词是 had -> `had had`)。
    2. 主句否定结构是 `would not have done` (not 紧跟第一个助动词)。
- **理解检查**:
    - **翻译练习**: "如果当时的启动资金没有那么高..." / "如果我想成为加盟商..." / "如果你之前跟其他的加盟商聊过..."
    - **学生的回应**: 能够自我修正 `had knew` -> `had known`；但在否定词序上仍需提醒。
    - **理解程度**: 中上 (逻辑懂了，肌肉记忆需加强)

### 问题 2: Tag Questions (反义疑问句)
- **学生的问题**: 
    1. 复杂主语的指代 (Sole traders -> They)。
    2. 否定前移 (I don't think...)。
    3. `Nobody` 作主语时的反义词。
- **初始理解**: 容易搞混 `doesn't` 和 `don't`；在 `Nobody` 句型中误用了否定尾巴。
- **给出的解释**:
    1. 复数名词 (Sole traders) -> 代词 `they` -> 助动词 `don't`。
    2. `Nobody` 含有否定义 -> 尾巴用肯定 `do they?`。
    3. `I don't think` 开头 -> 尾巴针对从句，且用肯定。
- **理解检查**:
    - **翻译练习**: "那位加盟商..." (he/doesn't he) / "没人想赔钱..." (do they) / "你并不认为..." (has it)。
    - **学生的回应**: 在提示下能纠正，掌握了 `Nobody` -> `do they` 的高阶用法。
    - **理解程度**: 高 (尤其是指代一致性)

# Part 2: 09BEC - Planning a Seminar (核心语言点)

## 涵盖的语言点 (Language Points Covered)

1.  **Take a [vehicle] straight to [place]** (直接搭乘...去...)
2.  **There's no point in [logical subject] doing sth** (某人做某事没有意义)
3.  **Confirm (that) / Confirm sth** (确认)

## 提出的问题与纠正 (Issues & Corrections)

### 1. 虚拟语气时态错误 (Subjunctive Past Tense)
-   **错误**: "If we **have** taken a taxi..." / "If we **have not have**..."
-   **纠正**: 虚拟语气过去时 (Type 3) 的 If 从句必须使用 **Past Perfect (过去完成时)**。
-   **正确结构**: `If + subject + had + past participle (done)`
-   **例子**: *If we **had** taken...* (不是 have taken)

### 2. There be 句型的反义疑问句 (Tag Questions)
-   **错误**: "There is no point..., **is it**?"
-   **纠正**: 当主句由 `There is/are` 引导时，反义疑问句的主语必须是 `there`，不能用 `it`。
-   **正确**: *There is no point..., **is there**?*

### 3. 性别指代一致性 (Pronoun Agreement)
-   **错误**: "Since **she** has quit..., what's the point in inviting **him**?"
-   **纠正**: 保持前后指代一致。
-   **正确**: *Since **he** has quit..., inviting **him**...*

### 4. 连词的使用 (Conjunctions)
-   **观察**: "In order to save time, we didn't check into the hotel, we just took a taxi..." (逗号连接两个独立句子，Run-on sentence)。
-   **建议**: 书面语中应使用连词 (but) 或分号 (;)。
-   **优化**: *...hotel, **but** we just took...*

### 5. 词汇辨析
-   **观察**: 把 "Office" (办公室) 翻译成了 "Airport" (机场)。虽不影响语法，但需注意听题准确性。
-   **观察**: 能够灵活使用 `What's the point in...?` 作为 `There's no point in...` 的疑问变体，非常地道。

## 今日亮点 (Highlights)

-   **Everyone 指代**: 完美掌握了 `Everyone has confirmed..., haven't they?` (前单后复) 的高阶规则。
-   **Confirm 用法**: 熟练使用 `confirm if/whether` 和 `confirm + noun phrase`。
-   **逻辑主语**: 成功构造了 `There's no point in both departments arguing` 这样复杂的带逻辑主语的结构。
