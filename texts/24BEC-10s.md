【学生说】我忘记了，这里 put 是过去分词，我以为是原型。

【老师说】太正常了！put 特殊在原形 / 过去式 / 过去分词同形，很容易混～
✅ 核心要点（一眼记牢）

	put 三态一致：put（原形）= put（过去式）= put（过去分词）
	本句 get put into 里，put 是过去分词，和 get 构成被动（不是原形）
✅ 快速判断技巧（不踩坑）看前面有没有 be/get：

	有 be/get → put 是过去分词（被动）：is put /get put
	无 be/get → put 是原形 / 过去式（主动）：I put...（现在 / 过去主动）
✅ 本句验证It gets put into... → get + put（过去分词）= 被动，表 “被纳入”
✅ 职场高频句（强化记忆）

	The task gets put on hold. 任务被搁置（put = 过去分词，被动）
	We put the task into the pipeline. 我们把任务纳入流程（put = 原形，主动）
要不要我帮你出 2 个小句子，你快速判断 put 是原形还是过去分词？

【学生说】那 it gets put into the development stream 这句话中的 development stream 是什么意思？

【老师说】development stream 职场里就是开发流程 / 开发流水线，和 dev pipeline 近义，更口语化

	整句：需求被纳入开发流程
	stream 表 “连贯推进的流程链路”，像水流一样从立项→开发→测试→上线顺走
✅ 职场高频搭配 & 同义替换

	同义：development pipeline（更正式）、dev workflow
	常用搭配：put into the dev stream（纳入开发流程）、follow the dev stream（走开发流程）
